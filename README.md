# Weather Radar

## Installation & running the app

```bash
$ git clone https://github.com/Hnes-co/Weather-Radar.git
$ cd Weather-Radar
$ npm install
$ npm run dev
```

## Adding your api key

- Inside Weather-Radar/src/ create api.json file with the following content:
```bash
{
    "apiKey": "your-api-key-here"
}
```
